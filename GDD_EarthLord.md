**版本**: 1.0

**游戏类型**: LBS末日生存游戏
**平台**: iOS

## 1. 游戏概念

### 1.1 一句话描述
> 在末日废土上，用真实的脚步丈量你的王国，圈地建家，探索求生。

### 1.2 游戏类型
- **主类型**: LBS（基于位置的服务）游戏
- **副类型**: 末日生存、模拟经营
- **灵感来源**: 《宝可梦GO》的LBS玩法 + 《饥荒》《Rust》的生存机制

### 1.3 核心玩法
**Walk to Earn** —— 通过真实世界的行走来获取游戏中的一切：
- 走路圈地 → 获得土地
- 走路探索 → 获得资源
- 走路交易 → 进行社交

## 2. 世界观与叙事

### 2.1 背景故事

#### 时间线

| 年份 | 事件 |
|-----|------|
| **2048年** | "终末战争"爆发，全球文明在72小时内崩塌，99%人口消失 |
| **2051年** | 幸存者从废墟中走出，《新开拓者协议》颁布 |
| **现在** | 玩家作为开拓者，重建文明 |

#### 核心法则《新开拓者协议》
> **"凡以双足丈量之无主土地，皆为开拓者所有。"**

就像小时候在沙滩上画圈占地——你拿着树枝，绕着走一大圈，圈起来的沙滩就是你的"领地"。现在，你的树枝变成了手机里的GPS，沙滩变成了整个城市。

### 2.2 视觉风格

**"希望朋克" (Hopepunk)**

| 元素 | 描述 |
|-----|------|
| 色调 | 废土灰褐 + 新生绿色 |
| 意象 | 废墟上的绿苗、破败建筑上的新标志 |
| 情感 | 绝望中的希望、重建的力量 |

### 2.3 叙事主题
这不是一个关于绝望的末日故事，这是一个关于**重建与希望**的故事——在废墟之上，用双脚开拓新的家园。

## 3. 核心循环

### 3.1 游戏循环图

```
┌─────────────────────────────────────────────────────────────┐
│                      游戏核心循环                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   登录注册 → 圈地占领 → 探索搜刮 → 获得资源                   │
│      ↑                              ↓                       │
│      │         ┌──────────────────────────────┐            │
│      │         │  资源用途：                    │            │
│      │         │  • 建造建筑（消耗资源）         │            │
│      │         │  • 交易换物（流通资源）         │            │
│      │         │  • 生存消耗（食物/水）          │            │
│      │         └──────────────────────────────┘            │
│      │                              ↓                       │
│      └──────── 通讯社交 ← 成就排行 ← 发展壮大               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 核心循环分解

| 阶段 | 玩家行为 | 游戏反馈 | 驱动力 |
|-----|---------|---------|-------|
| 圈地 | 户外行走画圈 | 获得领地 | 所有权欲望 |
| 探索 | 前往POI点 | 获得资源 | 收集欲望 |
| 建造 | 消耗资源 | 建筑物、功能扩展 | 成就感 |
| 交易 | 与玩家互动 | 获得所需资源 | 社交需求 |
| 生存 | 管理体征 | 保持活跃 | 紧迫感 |

## 4. 玩法系统详解

### 4.1 圈地系统

#### 4.1.1 基本规则

| 规则项 | 描述 |
|-------|------|
| 起始 | 点击"开始圈地"，开始记录GPS轨迹 |
| 过程 | 蓝色轨迹线实时绘制 |
| 闭合 | 回到起点50米范围内自动闭合 |
| 验证 | 系统检查合法性（无作弊、无重叠） |
| 完成 | 多边形变绿，领地归属玩家 |

#### 4.1.2 圈地限制

| 限制项 | 数值 | 说明 |
|-------|------|-----|
| 最小面积 | 100 m² | 防止刷地 |
| 最大面积 | 100,000 m² | 单次上限 |
| 移动速度 | < 30 km/h | 防止驾车作弊 |
| 轨迹间隔 | < 50米 | 防止跳点 |
| 每日次数 | 10次 | 免费用户限制 |

#### 4.1.3 领地属性

| 属性 | 描述 |
|-----|------|
| 名称 | 玩家自定义，如"优胜美地" |
| 面积 | 自动计算，单位m² |
| 建筑数 | 领地内建筑数量 |
| 活跃度 | 30天内需要活跃，否则回收 |
| 交易开关 | 是否允许100米内玩家看到挂单 |

### 4.5 交易系统

#### 4.5.1 交易流程

```
发布挂单 → 等待接受 → 确认交易 → 资源转移
    ↓           ↓
 取消挂单    超时失效(24h)
```

#### 4.5.2 交易规则

| 规则 | 描述 |
|-----|------|
| 可见范围 | 100米内可见对方挂单 |
| 领地开关 | 领地主可开关"允许交易" |
| 挂单上限 | 同时最多10个挂单 |
| 挂单有效期 | 24小时 |
| 交易手续费 | 无 |

#### 4.5.3 以物易物
游戏采用**纯物物交换**模式，无货币系统：
- "我提供": 选择要拿出的物品
- "我需要": 选择想要的物品
- 双方确认后资源自动转移

## 5. 经济系统

### 5.1 资源循环

```
探索搜刮 → 获得资源 → 消耗去向:
                      ├── 建造建筑
                      ├── 生存消耗（食物/水）
                      ├── 交易流通
                      └── 升级设备
```

### 5.2 资源产出与消耗平衡

**设计原则**:
- 日常探索可维持生存消耗
- 建造需要额外探索或交易
- 高级建筑/设备需要稀有资源（促进交易）

**数值参考**:

| 项目 | 日均产出 | 日均消耗 |
|-----|---------|---------|
| 食物 | 30-50 | 20（维持饱食度） |
| 饮用水 | 20-40 | 30（维持水分） |
| 木材 | 20-30 | 看建造需求 |
| 金属 | 10-20 | 看建造需求 |
| 稀有材料 | 1-3 | 看高级需求 |

### 5.3 免费与付费平衡

| 功能 | 免费玩家 | 付费玩家 |
|-----|---------|---------|
| 每日圈地 | 10次 | 无限 |
| 探索范围 | 标准 | 扩大50% |
| 背包容量 | 100格 | +200格 |
| 通讯设备 | 收音机/对讲机 | 快速解锁高级设备 |
| 生存衰减 | 标准 | 减缓30% |

## 7. 商业化设计

### 7.1 付费产品

**订阅制产品**

| 产品 | 价格 | 权益 |
|-----|------|-----|
| 开拓者月卡 | $9.99/月 | 无限圈地、探索范围+50%、衰减-30% |
| 领主通行证 | $19.99/月 | 开拓者月卡全部权益 + 专属皮肤 + 优先客服 |

**一次性购买**

| 产品 | 价格 | 内容 |
|-----|------|-----|
| 初级物资包 | $0.99 | 罐头×20, 饮用水×20, 木材×30 |
| 中级物资包 | $2.99 | 罐头×50, 饮用水×50, 金属×30, 药品×10 |
| 稀有资源包 | $9.99 | 卫星模块×5, 电子元件×20, 稀有材料×10 |
| 新手大礼包 | $4.99 | 对讲机×1, 初级物资包×2 |

### 7.2 付费点设计原则

1. **时间换金钱**: 免费玩家用时间获取，付费玩家用金钱加速
2. **不破坏平衡**: 付费不能获得免费玩家无法获得的核心功能
3. **外观分离**: 皮肤、称号等纯装饰性付费
4. **便利性付费**: 背包扩容、衰减减缓等便利性功能

## 8. 技术实现要点

### 8.1 反作弊系统

| 作弊类型 | 检测方法 | 处罚 |
|---------|---------|-----|
| GPS模拟器 | 检测模拟器App | 警告→封禁 |
| 速度异常 | 移动速度>30km/h | 轨迹无效 |
| 轨迹跳跃 | 相邻点距离>50m | 轨迹无效 |
| 频繁传送 | 短时间位置剧变 | 警告→封禁 |

### 8.2 性能优化

| 优化点 | 方案 |
|-------|------|
| GPS电量 | 后台低功耗模式，降低采样频率 |
| 地图加载 | 分块加载，缓存常用区域 |
| 排行榜 | 缓存机制，5分钟刷新一次 |
| 实时消息 | WebSocket长连接，心跳保活 |

### 8.3 架构模式

| 模式 | 应用场景 |
|-----|---------|
| 状态机 | 建筑状态、订单状态 |
| 观察者/事件总线 | 成就系统、实时更新 |
| 本地优先 | 背包数据、离线支持 |
| MVC/MVVM | UI架构 |
